## 5, LINUX
**Trong cửa sổ dòng lệnh (CLI), một số lệnh thông dụng mà chúng ta có thể sử dụng bao gồm:**

* `rm` + tên file ---> xoá file 
* `mkdir` + tên file ---> tạo thư mục ( nếu trong file đó có dữ liệu không thì chức năng đó sẽ xoá thư mục )
* `mv` ---> di chuyển và đổi tên
có 2 trường hợp 
1. `mv hello.c hello`
* trường hợp a : nếu **hello** là thư mục thì linnux sẽ di chuyển file **hello.c** vào bên trong thư mục **hello**
* trường hợp b : nếu **hello** không tồn tại thì linux sẽ đổi tên file từ **hello.c** sang h**hello**
* trường hợp c : nếu **hello** là file khác thì linux sẽ ghi đè nội dung của hello.c lên file hello đó
2. `mv hello.c hello/`
  * khảng định bảo linux chuyển file **hello.c** vào thư mục **hello**
 3. `mv+ tên file ..` là lên cấp bậc cao hơn
    vd mv hello.c.

    int n;
    while (true)

**đó là vòng lặp `while` ngoài ra ta còn vòng lặp `do-while`**
vòng lặp `do-while` 
```c

#include <cs50.h>
#include <stdio.h>

int main(void)
{
    int n;
    do
    {
        n = get_int("What's n? ");
    }
    while (n < 0);

    for (int i = 0; i < n; i++)
    {
        printf("meow\n");
    }
}
```
