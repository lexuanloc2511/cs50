## terminal or command-line interface ( CLI ) 
--> giao diện dòng lệnh là trong đó bạn nhập các lệnh 
hay **terminal window** : cửu sổ dòng lệnh đây sẽ là nơi bạn chạy các lệnh để biên dịch mã và ghi dữ liệu trong mã lập trình
## các lệnh Terminal đã học :
* `code hello.c`: dùng để tạo hoặc mở file code
* `make hello` : là người quản lý . nó ko **biên dịch** mà ra lệnh trình biên dịch dịch file này
* `./hello` : chạy chương trình đó ngay tại đây
```c
#include <stdio.h>

int main (void)
{
printf ("hello,word\n");
}
```
**graphical user interface**: giao diện người dùng độ hoạ (GUI)
* **library** là các mã máy do người khác viết đã đc biên dịch mà bạn có thể sử dụng
* **header files** là mục lục . bạn đọc nó để biết thư viện có gì và dùng thế nào
